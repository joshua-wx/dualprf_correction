dualprf_correction is a fork of [vcor_dual_prf](https://github.com/paltube/vcor_dual_prf) adapted for [pyodim](https://pypi.org/project/pyodim/) and trimmed down to the minimum code for the technique developed by [Hegstebeck et al., 2018](https://journals.ametsoc.org/doi/full/10.1175/JTECH-D-16-0230.1)

### Dependencies
- numpy
- scipy
- pyodim

### Notebook plotting Dependencies
- matplotlib

### Install from source
To install dualprf_correction, you can either download and unpack the zip file of the source code or use git to checkout the repository:

`git clone git@github.com:joshua-wx/dualprf_correction.git`

To install in your home directory, use:

`python setup.py install --user`
